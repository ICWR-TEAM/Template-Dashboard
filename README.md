# Dashboard Template incrustwerush.org

## Install

```bash
git clone https://github.com/ICWR-TEAM/Template-Dashboard
cd Template-Dashboard
npm install
```

## Run

```bash
npm run dev
```

## Build

Staging
```bash
npm run build:stag
```
Production
```bash
npm run build:prod
```
